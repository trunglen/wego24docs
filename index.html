Protocol Api

Protocol HTTP Request Public request bao gồm các request không cần xác thực như login, reset_password, … Private request bao gồm các request mà cần xác thực người yêu cầu. Ví dụ: update profile, đổi mật khẩu, thêm đơn hàng,… Tất cả private request cần gửi key kèm theo. Client nhận key ở trong response nếu thực hiện thành công một số api trong phần Authenticate. Wmap Api

Carrier Socket Connect to ws://<wmap_host>/wmap/carrier/join?access_token=<access_token> Example: ws://203.162.123.63:3002/wmap/carrier/join?access_token=MHJNaali5bJvbYb49jGRo97QQFSt636Kh9JRKlSWojW3vobWZVt4GN3QtlZLUUcvpmHCUN7w9USazD96 Echo (for Test/Ping) Request: /echo <data> Response: /echo <received data> Example: /echo 123 -> /echo 123 Set Carrier location type Point = {"lat": <float>,"lng":<float>,"time":<time stamp>} Request: /carrier/at <Array of Point> Các Point sắp xếp theo thứ tự tăng dần thời gian. Như vậy, Point cuối cùng chứa vị trí mới nhất Response: /carrier/at <len of recieved locations Example: /carrier/at [{"lat":10,"lng":20,"time":14235346456},{"lat":10.01,"lng":20,"time":14235346460}] -> /carrier/at 2
